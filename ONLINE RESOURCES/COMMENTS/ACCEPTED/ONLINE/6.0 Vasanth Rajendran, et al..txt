Academic Temporal Pattern Mining for Enhanced Student Performance Prediction: A Hybrid Machine Learning Approach

Os autores propuseram o framework Academic Temporal Pattern Mining (ATPM) para coleta de dados educacionais, incluindo registros de notas parciais e finais, frequência, dados demográficos, resultados em avaliações e participação em fóruns acadêmicos. A experimentação contou com uma amostra de 1000 alunos de graduação de diferentes cursos e a base de conhecimento foi organizada em janelas temporais de 12 semanas por semestre. O processamento foi realizado por um modelo de predição híbrido de Random Forest e XGBoost para identificação alunos em situação de risco. O sistema foi validado e apresentou uma acurácia de 87% e uma precisão de 80% a partir da sexta semana, permitindo intervenções pedagógicas tempestivas e alertas automatizados, fundamentados em dados significativos (p < 0.05). A solução projetada é caracterizada pela escalabilidade, pois adota práticas modernas de MLOps, utilizando Docker e Kubernetes para garantir a modularidade e o gerenciamento robusto do fluxo de dados coletados via REST API.
