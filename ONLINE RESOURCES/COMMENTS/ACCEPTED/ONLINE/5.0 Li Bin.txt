Cognitive Web Service-Based Learning Analytics in Education Systems Using Big Data Analytics

O objetivo do estudo é propor o Educational Clustering Big Data Mining System (ECBDMS), integrando-o a serviços web cognitivos (CWS-LA) para realizar a categorização segura de dados, facilitar a tomada de decisão e prever as qualidades e o desempenho acadêmico dos estudantes em sistemas educacionais digitais. O estudo utiliza um Algoritmo de Previsão de Caracterização e integra diversas técnicas de mineração de dados e aprendizado de máquina, como agrupamento K-means, Random Forest Regression, KNN (K-Nearest Neighbors) e Árvores de Decisão, todos operando em conjunto com serviços cognitivos web. O modelo ECBDMS apresentou resultados superiores a outros métodos, alcançando uma acurácia de previsão de 97,1%, um ganho de performance de 92,8%, uma proporção de predição de 88,6% e uma porcentagem de aprendizado de 94%, com uma taxa de erro de agrupamento de apenas 2,3%.
